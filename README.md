# Instructions

Review `diagram.drawio` with `drawio.com` for wiring instructions.

To build and upload new code to an Arduino:
``` shell
make find-dep  # this step is done once
make upload
```

To monitor the serial port:
``` shell
./scripts/serial_monitor.py
```

Enjoy!
